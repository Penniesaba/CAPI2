密码算法实现平台

提供多种密码学算法的Web实现，支持对称加密、哈希算法、编码算法和非对称加密等功能。

可通过Web界面使用，也可通过API接口调用。

支持的算法

对称加密算法

使用相同密钥进行加密和解密的算法。

  • AES (高级加密标准)
  • SM4 (国密对称加密)
  • RC6 (Rivest加密)
立即使用
哈希算法

将任意长度的数据映射为固定长度输出的算法。

  • SHA1 / SHA256 / SHA3
  • RIPEMD160
  • HMAC (消息认证码)
  • PBKDF2 (密钥派生)
立即使用
编码算法

数据格式转换算法，用于不同场景的数据表示。

  • Base64编码
  • URL安全的Base64
  • UTF-8编码
  • 十六进制编码
立即使用
非对称加密算法

使用公钥和私钥对的算法，支持加密和数字签名。

  • RSA (1024位)
  • ECC (椭圆曲线加密)
  • RSA-SHA1 (签名)
  • ECDSA (椭圆曲线签名)
立即使用
API接口示例

所有功能均可通过API接口调用，示例如下：


// SHA256哈希计算示例
fetch('/api/hash/sha256', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        message: "Hello, World!"
    })
})
.then(response => response.json())
.then(data => {
    console.log(data);
    // 输出: {
    //   "success": true,
    //   "data": {
    //     "hash": "dffd6021bb2bd5b0af676290809ec3a53191dd81c7f70a4b28688a362182986f"
    //   },
    //   "message": "SHA256哈希计算成功"
    // }
});
